# CSV Viewer

一个基于 Electron + Vite + React 构建的 CSV 数据查看工具，支持灵活的列配置、排序和拖拽调整列顺序。

## 功能特性

- 📊 **CSV 数据展示** - 支持大文件虚拟滚动
- 🔄 **多维度排序** - 点击列头即可排序，支持日期、数字、字符串智能排序
- ⚙️ **列配置** - 可显示/隐藏列，拖拽调整列顺序
- 📏 **行密度切换** - 紧凑/默认/宽松三种显示密度
- 🎨 **现代深色主题** - 美观的 UI 设计
- 📁 **拖放支持** - 直接拖放 CSV 文件到窗口

## 快速开始

### 方式一：使用启动脚本

```bash
chmod +x start.sh
./start.sh
```

### 方式二：手动启动

```bash
# 安装依赖
npm install

# 开发模式
npm start

# 仅启动前端
npm run dev

# 打包应用
npm run package
```

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **Vite** - 快速的前端构建工具
- **React** - UI 框架
- **PapaParse** - CSV 解析库
- **@dnd-kit** - 拖拽排序库
- **dayjs** - 日期处理库

## 界面预览

应用支持以下交互：

1. **打开文件** - 点击右上角「打开文件」按钮或拖放文件
2. **排序** - 点击任意列头进行升序/降序排序
3. **列设置** - 点击齿轮图标打开列配置面板
4. **密度切换** - 使用工具栏中的密度按钮切换行高

## 数据格式

支持标准 CSV 格式，首行为表头。示例：

```csv
Date,Kind,Model,Cost
2025-12-04T11:45:59.432Z,Included,composer-1,0.02
```
